secrets 
object type 
securely stores a piece of information in the cluster such as a db password,
api key, ssh key

imperative command to create, needs to be created manually

creating a secret:
kubectl create secret generic <secret_name> --from-literal key=value

kubectl create secret generic pgpassword --from-literal PGPASSWORD=PASSWORD90

different types of secrets:
generic <--- mostly using
docker-registry
tls

kubectl get secrets

- name: PGPASSWORD
              valueFrom: 
                secretKeyRef:
                  name: pgpassword <--- name of secret
                  key: PGPASSWORD <--- dictionary key
